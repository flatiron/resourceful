resourceful by flatiron

A storage agnostic resource-oriented ODM for building prototypical models with validation and sanitization

Installation

Server-side

npm install resourceful

Browser-side

Download one of these browser builds (you can build one yourself by executing bin/build in Resourceful source tree). Put it anywhere you want and just source it in your HTML (you can also hotlink it from this page if you feel like).

  • resourceful@0.1.5

See example of Resourceful use in the browser.
